# OpenPifPaf: Performance vs Efficiency<br />

## Training: 
To train the Openpifpaf model, you mgiht use the follwoing command:<br />
python train.py<br />

Options:<br />
Learning rate: --lr<br /> 
Learning rate decay: --lr-decay<br />
Momemtum: --mometum<br />



Evaluation: python eval.py<br />
